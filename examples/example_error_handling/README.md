# example_error_handling

What it demonstrates: Global error handler, throwing in handlers and middleware.

## Features
- `app.onError` — catches all unhandled exceptions
- `app.notFound` — custom 404 handler
- `GET /throw-handler` — exception thrown inside a handler
- `GET /throw-middleware` — exception thrown inside middleware

## Run
```bash
dart run bin/main.dart
```
